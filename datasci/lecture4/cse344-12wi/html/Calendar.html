Readings

Lectures 1-3: Introduction, Data Models, and Simple SQL

  • SQL for Nerds, by Philip Greenspun, at  http://philip.greenspun.com/sql/ 
  • Data types, data models, data management systems (Sections 2.1, 2.2, and 2.3)
  • Simple Queries in SQL (Section 6.1)
  • Queries with more than one relation (Section 6.2)
  • Nulls (Sections 6.1.6 - 6.1.7)
  • Outer joins (Section 6.3.8)
  • SQL Lite Documentation, http://www.sqlite.org/lang.html

Lectures 4-7: SQL and Basic Physical Tuning

  • Subqueries (Section 6.3)
  • Aggregations (Sections 6.4.3 - 6.4.6)
  • Basic Physical tuning: Indexes (Sections 8.3 and 8.4)

Lectures 8-11: Relational Algebra, Relational Calculus, Datalog

  • Relational algebra (Sections 2.4, 5.1 and 5.2)
  • Relational query execution (Sections 15.1 through 15.3)
  • Relational Calculus and Datalog (Sections 5.3 and 5.4)
  • Three Query Language Formalisms, a brief tutorial on these three languages.

Lectures 12-13: XML/XPath/XQuery

  • Semistructured data and XML (Sections 11.1 and 11.2)
  • DTDs (Section 11.3)
  • XPath (Section 12.1)
  • XQuery (Section 12.2)

Lectures 14-17: Database Design

  • E/R diagrams (Sections 4.1- 4.4) [Old edition: Chapter 2]
  • From E/R diagrams to relations (Sections 4.5 and 4.6) [Old edition: Sections 3.2 and 3.3]
  • Constraints (Sections 2.3, 7.1, 7.2) [Old edition: Sections 7.1 and 7.2 only]
  • Design Theory (Sections 3.1, 3.2, 3.3)
  • Views (Sections 8.1, 8.2, 8.3) [Old edition: Sections 6.6 and 6.7]

Lectures 18-20: Transactions

  • Transactions in SQL (Section 6.6)
  • Transactions and concurrency control (Sections 18.1 through 18.3)
  • We also covered some content that appears in Sections 19.1 and 19.2

Lectures 21-24: Parallel Databases

  • Definition and basic concepts in parallel databases (Section 20.1)
  • Map-reduce (Section 20.2); Chapter 2 (Sections 1,2,3 only) of Mining of Massive Datasets, by Rajaraman and Ullman (optional: the original MapReduce paper)
  • Pig latin: a not-so-foreign language for data processing. Christopher Olston, Benjamin Reed, Utkarsh Srivastava, Ravi Kumar, and Andrew Tomkins. SIGMOD'08.
  • Related to hw6. Optional reading on the statistical properties of the Web: Ravi Kumar, Prabhakar Raghavan, Sridhar Rajagopalan, D. Sivakumar, Andrew Tomkins, Eli Upfal: The Web as a Graph. PODS 2000: 1-10):

Lectures 25: Data integration and data cleaning

  • Data integration (Section 21.1 and 21.2)
  • Data cleaning (Section 21.7)

Optional, extra readings

There will be no exam questions on the following book chapters, but you may still find them interesting to read now that you have mastered the basics of data management

  • SQL in a Server Environment (Chapter 9): We discussed some of the basic ideas in lecture and you saw this in action through assignments. The chapter covers in greater details how DBMSs are typically deployed and accessed from applications.
  • Advanced topics (Chapter 10): All very useful stuff too that we simply do not have time to cover.
